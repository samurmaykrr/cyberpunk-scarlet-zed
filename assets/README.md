# Assets Directory

This directory contains preview images and screenshots for the Cyberpunk Scarlet theme.

## Required Images

- `preview-dark.png` - Screenshot of the dark theme variant
- `preview-light.png` - Screenshot of the light theme variant

To add screenshots:
1. Take screenshots of the theme in Zed
2. Name them according to the convention above
3. Add them to this directory
4. Update the README.md to reference them
